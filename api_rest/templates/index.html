API de Gerenciamento de Produtos

Esta é uma API Django desenvolvida para gerenciar produtos em um sistema de estoque.

Endpoints

  • Listar todos os produtos: /api/product/
  • Obter detalhes de um produto por ID: /api/product/<id>/
  • Criar ou excluir um produto: /api/data/<id>/

Para mais detalhes sobre cada endpoint e os métodos HTTP suportados, consulte a documentação da API.
